# GUIFly

GUIFlyはMinecraftのGUIを操作するためのライブラリです。<br/>

できること

・GUIのアイテムを変更する<br/>
・GUI内のアイテムのクリック/ピックアップ/移動/変更の検出<br/>
・GUIの種類を変更する<br/>
・GUI内の範囲選択<br/>
・GUIのタイトルを変更する<br/>
・GUIの種類に応じたイベントの受け取り(WIP)<br/>
